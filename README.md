openNetworkHMI
=======

Web based framework for creating HMI interfaces.

INSTALL
=======

1) git clone --recursive https://github.com/matmir/openNetworkHMI.git

2) Go to openNetworkHMI directory and run ./install.sh

Project site: https://opennetworkhmi.net
